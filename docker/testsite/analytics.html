Google Analytics 4 analytics

GA4 sets these cookies when loaded:

  • _ga — Client ID (2 years)
  • _ga_* — Session data (2 years)
  • _gid — Session ID (24 hours)
  • _gat — Throttle requests (1 minute)

GA4 script tag (measurement ID: G-XXXXXXXXXX)

Uses a dummy measurement ID so it won't actually send data to Google,
but the gtag.js library will still set _ga and _gid cookies.

Hotjar analytics

Hotjar sets these cookies:

  • _hj* — Various Hotjar tracking cookies
  • _hjSessionUser_* — Hotjar user ID
  • _hjSession_* — Session data
